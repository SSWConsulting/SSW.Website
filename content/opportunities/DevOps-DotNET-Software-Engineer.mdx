We are a leading software consultancy firm in France, Australia and China. We are seeking a skilled DevOps & .NET Software Engineer to join our team based in Strasbourg, France.

As a DevOps & .NET Software Engineer, you will be responsible for developing and maintaining our software solutions while implementing modern DevOps practices to enhance the software development lifecycle.

This role requires great communication skills and solid experience in delivering production-ready web solutions. Further, we encourage you to take ownership of your relationship with clients and build meaningful connections, think critically, and create the right solution for the right job.

#### Role Responsibilities

* Design, develop, and maintain software applications using .NET technologies, ensuring high performance, scalability, and reliability.
* Implement DevOps practices and tools to automate software builds, testing, and deployments.
* Collaborate with cross-functional teams to define software requirements and ensure the successful delivery of projects.
* Troubleshoot and resolve software defects and issues, working closely with the stakeholders.
* Continuously monitor and optimize software performance, identifying areas for improvement.
* Participate in code reviews to ensure adherence to coding standards and best practices.
* Collaborate with the infrastructure teams to manage and maintain various environments (e.g., dev, UAT, and production)
* Stay updated with the latest trends and advancements in DevOps practices, .NET technologies, and related tools.

#### Candidate Profile

* Excellent communication skills (English and French)
* An IT degree (or equivalent experience)
* 3+ years of experience as a DevOps Engineer / Software Engineer with a focus on .NET development
* Proficiency in .NET technologies, including C#, ASP.NET, and .NET Core.
* Databases - SQL Server, CosmosDB
* Frontend – SPAs such as React, Angular, and Blazor
* Good understanding of Agile methodologies and the software development lifecycle.
* Ability to work effectively in cross-functional teams
* Solid understanding of DevOps principles and practices, including CI/CD, and infrastructure as code
* Experience with configuration management and deployment tools such as Azure DevOps, GitHub, Octopus Deploy, etc.
* Knowledge of containerization technologies like Docker and orchestration tools such as Kubernetes
* Strong scripting and automation skills using PowerShell, and Azure CLI
* Familiarity with cloud platforms like Microsoft Azure, AWS, or GCP
* Excellent problem-solving skills
