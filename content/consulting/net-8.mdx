# Why .NET 8?

.NET 8 is the latest LTS (Long Term Support) version of the platform – migrating now will futureproof your applications.

.NET 8 provides robust enterprise functionality and cloud-native development all within a stable ecosystem backed by Microsoft.

For your team, .NET 8 is the ultimate dev experience, updated with state-of-the-art features like C# 12, Native AOT (Ahead of Time Compilation), and source code generation.
