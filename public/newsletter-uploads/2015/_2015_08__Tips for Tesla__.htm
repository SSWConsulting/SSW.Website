"Enterprise Angular, Javascript and Hangfire in the Aussie Government" – Nicholas Campbell and Alexander Kachalkov

  • Javascript architecture CASA has utilized within its Medical Records System update project
  • In-depth look at core.glue; Exposing dotNet Enums, Consts and Classes to the Javascript frontend auto-majicially.

Alexander Kachalkov will provide a quick introduction of running background jobs using https://hangfire.io:

  • An easy way to perform fire-and-forget, delayed and recurring tasks inside ASP.NET applications. No Windows Service required.
  • Backed by persistent storages. Open and free for commercial use.
  • Fire-and-forget, delayed, recurring tasks.
  • Run inside a web app or anywhere else.
  • Automatic Retries
  • Integrated Monitoring UI
