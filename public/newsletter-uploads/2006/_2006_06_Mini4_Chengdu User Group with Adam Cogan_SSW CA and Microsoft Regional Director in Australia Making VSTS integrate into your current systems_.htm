使用VSTS优化软件开发生命周期

Visual Studio Team System (VSTS) 和 Team Foundation Server (TFS) 结合在一起，可以对软件产品开发的整个周期进行全面的支持，包括：最初设计，测试和开发过程；并且还集成了对任务跟踪和项目管理的支持。这次的演讲内容将带你通过VSTS的全新功能对整个产品开发周期进行更多的了解：

  1. 任务跟踪管理系统应该如何工作？
  2. TFS work items系统如何工作？
  3. 现有的系统和TFS系统缺乏有效的集成。
  4. 如何定制TFS系统的功能
  5. 如何把TFS work items集成到现有的任务跟踪管理系统中
  6. 介绍其他相关的第三方工具
