Office DevCon 2007

22 days to go!

Office DevCon 2007
Saturday 3rd November 2007 -
Sunday 4th November 2007
Microsoft office, North Ryde

This should be a good event, and it also includes our Adam Cogan who will present 3 topics:

  • My Favourite 3rd Party Office Utilities
  • Using Office 2007 and SharePoint 2007
  • Collaboration Without Borders: Office Enterprise

Office DevCon 2007 is a world-class, annual community conference squarely aimed at Office developers and power users – not information workers. Over 2 days it will cover Office core technologies for both 2003 and 2007 such as Access, Word, Excel, PowerPoint, Publisher, Groove, InfoPath, OneNote and Outlook.

It will also be covering SharePoint (specifically as a standalone product, and how to build collaborative Office applications), SQL Server 2000 and 2005 (specifically how to integrate Office applications with SQL Server), Visio, Project, Internet Explorer, Microsoft Dynamics and CRM. They also have advanced courses on VSTO and how to create managed Office-based applications (for both Office 2003 and 2007), Database design and data modelling, Systems design and modelling, VBA code patterns, Business Intelligence (Office-based) and anything else that may be of interest to Office developers and power users for those who cant get enough.

Sydney .NET User Group -
The best place to learn .NET for free in Sydney

5 days to go!

Sydney .NET User Group
Wednesday 17th October 2007
5:45pm - 9:00pm
Microsoft, Sydney

Taking Your Application Design to the Next Level With Data Mining

Data mining as a technology is designed to analyze large volumes of data looking for patterns that accurately predict behaviour. These patterns can then be used to derive knowledge about data, and in turn this knowledge can be used to enhance application designs and the user experience.

In this presentation, Peter Myers will present an overview of SQL Server 2005 Data Mining and will demonstrate how to develop data mining models that can be embedded into your applications. Demonstrations will be based on SQL Server 2005 Data Mining, using .NET Windows and web development.

This presentation is a must for any developer looking to embed “Artificial Intelligence” into their solution design to take their applications to the next level. It is guaranteed to thrill you with potential, and excite you with the ease in which it can be accomplished. Note that there is no requirement that attendees have any data mining background.

Peter Myers is a freelance BI practitioner, trainer and presenter. He has 11 years' experience working in OLTP database design and development, DBA support and Data Warehousing/Business Intelligence with SQL Server. Peter has a broad business background supported by a bachelor degree in applied economics, and he extends this with MCITP, MCSD, MCDBA, MCSE and MCT certifications. Currently he provides mentoring and technical training locally and internationally, and focuses on delivering training for the SQL Server database engine, Integration Services (DTS), Analysis Services, Reporting Services, ASP.NET and .NET programming (C# and VB.NET).
