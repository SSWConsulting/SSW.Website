The Model View Controller design pattern has been an proven and effective way of structuring applications for more than 30 years. From Zero to Everywhere with ASP.NET MVC 4

Mobile devices like tablets and phones are quickly becoming the primary way people access the web. So why do so many websites still provide a horrible mobile experience?

With ASP.NET MVC, it's easier than ever to literally put your site in the palm of everyone's hand. MVC 3 had many great features that allow you to target mobile devices with different layout and even content. The recent releases of MVC 4 makes things even easier and show that Microsoft is really thinking about web-enabled devices and new web technologies.

In this demo-heavy session, Damian Brady will show off some of the best parts of ASP.NET MVC 4 to help make targeting mobile devices easy.

Introduction to Windows Phone 7 development

Have you ever wanted to have a look at how exactly you can go about creating apps for Windows Phone 7? Then this session is for you!

In this session Brian will discuss how he learnt the ropes for WP7 app development and will use his app ‘Stitch’ as a real example (http://stitch-app.com/). Throughout the session Brian will discuss how the application was structured using the MVVM model and the Phoenix framework, how authentication is handled using OAuth and how RestSharp and AsyncCTP3 are used to make calls to secured data sources returning JSON data.

We will also cover some Windows Phone specific items such as how to set live tiles for your application, make use of the camera and isolated storage so that your apps can really shine. Lastly the session will include some info on the app submission and verification process, including some tips on how you can ensure that your apps pass submission first time.

If you are interested in app development for WP7 then this session is a must see!
