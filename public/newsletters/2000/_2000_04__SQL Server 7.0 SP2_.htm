5. Hot Tips

· The StrConv Function
· Simplifying Debugging


1) The StrConv Function
If you have problems with the formatting of sting data, the StrConv function can be useful.
For example, you may wish to convert a name of "JOE BLOGGS" to proper case i.e "Joe Bloggs", or you may wish to convert from lower to upper case.
For example, to convert all data in a field to proper case, you could use:

StrConv([User_Name],3)

or to convert to upper case, you could use:

StrConv([User_Name],1)

Thanks to David Klein for this tip

2) Use Resume 0 to simplify debugging.

If you get an error, hit Ctrl-Break at the error msgbox and the code window appears with the msgbox command highlighted. Place the cursor on the Resume 0 and hit Ctrl-F9 (set next line) then Ctrl-F8 (step one line) and you are back at the line that caused the error. I automatically add "Resume 0 'For debugging" to the end of all error handlers for this purpose.

Thanks to Ted Avery for this tip
