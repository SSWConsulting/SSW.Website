The FireBootCamp program is an intensive, full-time, 9-week program. Participants learn, work and code together for 10-12 hours per day, 5 days a week. There are exercise and food breaks, but for the main the program is continual hands-on learning.

Get Agile with 3 Day Scrum Training!

Intensive Professional Scrum Developer Course

Brisbane, Melbourne, Sydney, Perth, Canberra, Adelaide

Cost: $2,995 + GST

Professional Scrum Developer courses teach students how to work in a team - using modern software engineering practices - to develop an increment of potentially releasable functionality using a specific technology platform.

Classes are exercise-driven, with students working in self-organizing teams to develop "done" increments from Product Backlog items.

Course at a Glance:

This 3-day Professional Scrum Developer .NET course is a mix of lecture, demonstration, group discussion, simulation, and hands-on software development. "Activating your MSDN Azure Benefits" - Chris Birmele

Bring all your cloud questions to this session! Did you know that MSDN subscribers have Azure credits that they can use each month to deploy virtual machines, websites and cloud services? In this interactive session, Chris Birmele will give an overview of the services available with Microsoft Azure and a walkthrough of how to activate your MSDN benefits. You’ll see demos on how fast and easy it is to create your own development environments in the cloud, including: web sites, VMs, automating with PowerShell, connecting to on-premises services with Service Bus and load balancing cloud services.

About the presenter:
Chris Birmele is a Product Technology Specialist for Microsoft Australia, focusing on Visual Studio, TFS, Release & Test Manager and increasingly all things Azure. He has 20+ years’ experience with Microsoft, IBM, Rational Software, Compuware in a variety of technical roles. Originally from Zurich, Switzerland he enjoys travelling, photography and motorcycles.
