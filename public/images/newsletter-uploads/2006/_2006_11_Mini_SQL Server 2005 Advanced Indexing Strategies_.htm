Tech Breakfast - SYDNEY: SQL Server 2005: Advanced Indexing Strategies

2 DAYS TO GO!

SQL Server 2005: Advanced Indexing Strategies
Friday 15th December 2006
8:00am - 11:00am
Location: SSW
 $150 (+GST)


In this session, Adam Cogan will take you through a fast-paced collection of indexing tips and tricks based on an already optimised base table structure. How you create your base table and choose the clustered index has a big impact on that table's performance. Once the base table structures have been defined; you still have to further improve system performance. Where do you look and how do you do this?

Adam will talk about targeting both the known and unknown performance problems. Once you know where the problems are, he will go through numerous options to improve the performance using indexes.
