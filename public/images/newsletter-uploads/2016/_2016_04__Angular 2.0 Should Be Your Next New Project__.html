Angular 2.0 Should Be Your Next New Project
SSW has had great success with Angular 1.x and we think it is an excellent platform for building modern web applications. In case you haven't built a website or looked for a job in a few years, Angular is everywhere.
While there are a few other popular JavaScript frameworks for building websites and web applications, none come near to the popularity of Angular.

Figure: Angular is the number #1 JavaScript framework, with only React even registering as competition.
While Angular 1.x has been massively popular, it is now 6+ years old and like any software project that runs for that long, a re-write can include some pretty significant improvements. We were very happy with Angular, but we also had a few issues that we wished they would address.
This is exactly what the team at Google have done with the next version of Angular a.k.a. Angular 2.0.

April User Group - Sydney
Performance Tuning from the Trenches
Performance is often the last thing that developers think about when working on their projects. Often times it is after the website has crashed and burned before performance gets brought to the top of the backlog.
In this session Eric will share war stories from helping clients performance tune their applications and guide you through the process of:
  • Identifying the bottle necks
  • Finding solutions to the bottlenecks whether it be:
    • Code changes
    • Hardware changes
  • Setting up a process to continually measure performance
About the presenter
Eric Phan is a Sydney-based developer and is one of 2 Chief Architects at SSW. He has an in-depth knowledge of most aspects of .NET and is an advocate of coding standards compliance and unit testing. When he's not immersed in the world of code, he enjoys spending time with his wife and 2 children.
