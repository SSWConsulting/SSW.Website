This month - Visual Studio.Net

Visual Studio.Net is the next generation of development tools from Microsoft, replacing Visual Basic and Visual Interdev. It will reduce the number of situations where Access will be the chosen developer tool.

This session by Stephen Jeffries from Microsoft and Adam Cogan will provide an overview of the Microsoft .NET Platform, and services, including an overview of the Common Language Runtime, and .NET Base Services. We will also have a look at Visual Studio .NET for building a Web Enabled Application, using ASP.Net, C# (C Sharp), and Visual Basic.NET. Adam says that the example will give you a good idea of what a web service is and how to build one.
