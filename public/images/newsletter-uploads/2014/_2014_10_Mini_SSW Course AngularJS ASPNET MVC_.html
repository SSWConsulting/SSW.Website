Course overview

Looking to build the best modern apps? Step into the future with SSW and learn Modern Enterprise Best Practices with AngularJS and ASP.NET MVC. In this 2 day, Microsoft-approved course you will learn proven tools and techniques for building enterprise-quality, maintainable, scalable, and robust applications.

Key concepts:

  • Dependency injection
  • Loosely coupled architecture
  • SOLID principles
  • Azure design patterns
  • Repository unit of work

Further learning items will include:

  • Overview of Azure, including
    • Azure websites
    • Azure SQL database
  • TFS Builds & Continuous Integration
  • Enterprise Architecture
  • Enterprise MVC Best Practices
  • Implementing SPAs in the Enterprise

Course runs in Sydney on the 2 days immediately following TechEd, on October 29 and 30. Seats are limited so book now to avoid disappointment.
