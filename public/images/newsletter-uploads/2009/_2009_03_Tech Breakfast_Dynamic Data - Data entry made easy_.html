Tech Breakfast on Dynamic Data
 
 
Create Data Driven Websites Without Writing Code by Eric Phan
 

Dynamic Data - Data entry made easy with ASP.NET

Watch a live demonstration and see how to build an awesome website using tips and tricks in Visual Studio.Net, ASP .NET and J-query

Learn how to create data driven websites in 5 minutes from SSW real life case studies

Enjoy a fulfilling healthy breakfast

Date: 27 March 2009
Time: 8am - 11am

When working with databases, we often need to create some administrative pages to do all our CRUD (create, read, update and delete) operations on the tables. This can be a long and repetitive process if you don't use a code generator. Thankfully in Visual Studio 2008 SP1, Dynamic data allows us to create all those pages without writing a single line of code!

In this session you will also find out how to customize these admin pages with different looks and data entry controls.
