UTS Short Course - SQL Server 2012 for Developers

UTS - SQL Server 2012 for Developers
Every Tuesday
Dates: March 5th - April 2nd
Times: 6:00-9:00pm
Duration: 5 weeks
Venue: City - Broadway
Price: $800

More Information

This is the 1st university course in the world to teach students on SQL Server 2012. It will provide attendees with the opportunity to take advantage of the new features provided by SQL Server 2012 so that they can design, build, query and develop enterprise SQL Server 2012 databases and servers.

This is a practical course for the ICT Industry utilising the latest SQL Server 2012 developed by Microsoft. SQL Server 2012 provides much anticipated improvements to the management studio with the move to Visual Studio shell, as well as new data quality tools and a wealth of programming enhancements. This course provides attendees with the knowledge necessary to take advantage of new features provided by SQL Server 2012. It provides a comprehensive overview of technologies, security and design best practices with particular emphasis on hands-on skills.

UTS Short Course - Developing Windows and Web Applications with MVC4 and Visual Studio 2012

UTS - Developing Windows and Web Applications using Visual Studio.NET
Every Thursday
Dates: March 7th - May 16th (excluding April 25th)
Times: 6:00-9:00pm
Duration: 10 weeks
Venue: City - Broadway
Price: $1,600

More Information

UTS has the proud distinction of being the first academic institution in the world to teach participants to develop on the .NET Framework with Visual Studio 2012. The .NET Framework is one of the leading Microsoft development platforms for Windows and Web applications and this short course aims to introduce attendees to the latest Microsoft Development platform Visual Studio 2012.

The framework improves on Java's capabilities, providing an API for multiple languages to generate and communicate with each other through the Common Language Runtime(CLR). Conceptually, .NET languages are not much different to traditional OO languages, however, students and programmers may need a short training course to understand its capabilities, the functionalities, the object and communication structure for these services and the best practices surrounding these technologies.

In this course, participants will be exposed to C# as well as learn how to rapidly create data driven forms on the web and on windows.

UTS Short Course - SQL Server 2012 for Business Intelligence

UTS - SQL Server 2012 for Business Intelligence
Every Tuesday
Dates: April 16th - May 14th
Times: 6:00-9:00pm
Duration: 5 weeks
Venue: City - Broadway
Price: $800

More Information

This is one of the first academic course in the world to teach students on SQL Server 2012 for Business Intelligence. SQL Server 2012 blurs the lines between relational and multidimensional databases. Business Intelligence is about the move from storing data to retrieving information from your database systems.

Microsoft SQL Server 2012 is a complete business intelligence (BI) platform that provides the features, tools, and functionality to build both classic and innovative kinds of analytical applications.

Microsoft SQL Server 2012 Business Intelligence components supports the development and use of business intelligence in enterprises of all sizes. It can be used by employees and external parties, not just management and analysts. To this end, SQL Server 2012 is complete, integrated, easy to use, publishes data as web services, delivers excellent performance on commodity hardware and contains many new features that can be used to develop innovative analytical applications.
