This month - Developing for Microsoft SharePoint Portal (Tahoe) Server and the Web Storage System

SSW has been using Tahoe for a couple of months now and we think it's great. Adam Cogan will introduce you to the ins and outs of Tahoe including how it can help with both enterprise portal/search as well as document management solutions. You'll learn how Tahoe can provide for relatively simple development of departmental Web Storage System solutions. All the while, he'll illustrate these key features with live demonstrations, examples and code!
